# `@humanoids/eslint-config`

> The humanoids [Eslint](https://eslint.org/) config.

## Usage

**Install**:

```bash
$ npm i --dev eslint

$ npm i --dev eslint-plugin-react-hooks

$ npm i --dev @humanoids/eslint-config
```

**Create `.eslintrc.js`**

```jsonc
module.exports = {
  extends: "@humanoids/eslint-config"
};
```
